# Vue-shiyanlou 后端转发

实验楼的API转发。

# 线上使用
```
http://139.129.89.253/api
```

# 技术栈
```
Django + requests
```

# 安装
```
pip install Django
pip install requests
pip install django-cors-headers
```

# 运行（本地服务器）
```
python manage.py runserver
```
